If your build-up breaks under pressure, your transitions usually break right after it.

I just finished a data-forward soccer analysis project that tracks two linked failure modes: build-up collapse in your own half and post-regain waste in the next few seconds.

The work uses StatsBomb Open Data for event-level sequence logic, ASA for league-context aggregates, and a reproducible Python pipeline so every chart can be traced to code and source files. I kept the framing narrow on purpose: this is an associational profile, not a causal claim.

One finding I think is worth discussing: in the Inter Miami 2023 Open Data sample, 319 of 353 tracked regains (90.4%) did not produce a modeled shot inside the project’s short post-regain window. That signal sits alongside uncertainty bands and explicit sample-size caveats, so it reads as a pattern to investigate, not a verdict.
